Projects

Conversession

A secure and feature-rich video conferencing web application built with JavaSE, Spring Boot, HTML, CSS, JavaScript, and PostgreSQL. Key features include user authentication, real-time audio/video communication, global chat, and scalable backend support for seamless meeting experiences.

Yoga Steps Prediction

The Yoga Steps Detection and Prediction System uses Python, Machine Learning, and SQLite to identify eight yoga poses in real-time via webcam. It achieves 95% accuracy using OpenCV and MediaPipe for pose recognition and includes a secure user login system. Designed to help practitioners and instructors monitor and enhance yoga practices.

Colmar-Academy

This project involved designing a professional landing page for Colmar Academy, a new school. Provided with wireframes and media resources, all other design decisions were ours. It focused on responsive design techniques, including media queries, to ensure a seamless experience on both desktop and mobile devices.
